# group

 ``` javascript
 <group>
    <cell title="title" value="value"></cell>
  </group>
 ```

#####属性
---
名字         | 类型            | 默认值     | 说明                                   | 版本要求
---          | ---             | ---       | ---                                    | ---
title        | String          |           | 分组标题                                | -- 
label-width  | String          |           | 为子元素设定统一label宽度(例: 80px)       | --
